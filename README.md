# バイトシフトのアプリ（現在リファクタリング中）

## アプリケーション概要
自身のパソコンでgoogle calendarに登録できるようする

## アプリURL（デプロイ先）
https://baito-shift-app.vercel.app

## 使用技術
* フロントエンド: React, TypeScript
* バックエンド: Spring Boot, Java 
* データベース: MySQL
* インフラ・その他: 

## ✨ 主な機能
* 自身のgoogle calendarへのログイン機能
* アルバイトの仕事内容、時間、一緒に働く人の名前などの登録
* データベースへの登録

## 💡 開発の背景・目的
自分のアルバイト先のシフトが紙で張り出されるため、毎回手入力するのに時間がかかったり、間違えて入力することも多かった。

## 🔥 工夫した点・苦労した点
* フロントエンドでTypscriptでの開発は初めてであったため、Geminiに相談しながら進めた
* **パフォーマンスの改善:** 〇〇のデータ取得が遅かったため、〇〇という技術を使って読み込み速度を改善しました。
* **UI/UXのこだわり:** 中央に登録するボックスを配置したり、仕事のタスクと仕事時間が同じである場合は自動入力できるようにした(後ほど編集もできるようにした)。
